### Step 1.3 ‚Äî Deploy to Vercel

1. Go to **https://github.com** ‚Üí click **New Repository**
2. Name it `landing-page` ‚Üí click **Create Repository**
3. Upload your `index.html` file to the repo
4. Now go to **https://vercel.com/dashboard**
5. Click **Add New ‚Üí Project**
6. Select your `landing-page` GitHub repo
7. Click **Deploy**
8. In ~30 seconds you'll get a live URL like:
   `https://landing-page-abc123.vercel.app`

‚úÖ **Save this URL ‚Äî this is what you'll send people in your Instagram DM**

## üîß PHASE 2 ‚Äî Build the n8n Workflow

Now go to your n8n dashboard and create a **New Workflow**. Name it: `Instagram Lead + Email Sequence`

---

### Node 1 ‚Äî Webhook (receives form data)

1. Click **+** ‚Üí search **Webhook**
2. Set **HTTP Method** to `POST`
3. Click **Listen for test event** (keep this tab open)
4. Copy the **Webhook URL** shown
5. Go back to your `index.html` ‚Üí replace `PASTE_YOUR_N8N_WEBHOOK_URL_HERE` with this URL
6. Now submit your landing page form once with test data
7. n8n will show you the incoming data ‚Äî click **Use for testing**

---

### Node 2 ‚Äî IF (validate the data)

1. Add node ‚Üí search **IF**
2. Add 3 conditions (use AND):
   - `{{ $json.name }}` ‚Üí **is not empty**
   - `{{ $json.email }}` ‚Üí **is not empty**
   - `{{ $json.consent }}` ‚Üí **is equal to** ‚Üí `true`
3. The **true** branch continues the workflow
4. The **false** branch ‚Üí leave it (or add a Stop node)
